# DevSecOps Project - Sécurité Appliquée

## Description
Projet académique de démonstration d'un pipeline CI/CD sécurisé avec intégration complète d'outils DevSecOps.

## Architecture
- Pipeline Jenkins avec 14 étapes de sécurité
- Analyse SAST, SCA, DAST automatisée
- Détection de secrets avec Gitleaks
- Évaluation quantitative des risques
- Conformité OWASP ASVS, PCI-DSS

## Technologies
- Jenkins, SonarQube, Trivy, Gitleaks, OWASP Dependency Check
- Docker, Kubernetes
- Python, Bash, Groovy, YAML
